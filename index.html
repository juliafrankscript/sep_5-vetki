Ветка Git - это версия проекта. GIT HAB-площадка для размещения ssh ключей(публичных) GIT HAB поддерживает возможность генерации статических сайтов Создание веток git branch name; Удаление веток git branch -D name; Переключение с ветки на ветку git checkout name; Загрузка веток на GIT HAB git push -u origin name; Удаление ветки с GIT HAB git push -d origin name; Публикация статического веб-сайта с помощью GIT HAB PAGES открываем Your repo,далее тот репозиторий,который хотим опубликовать в виде сайта, далее настройки Settings,вкладка Pages,Branch Master,сохранение Save,ожидаем ссылку.
